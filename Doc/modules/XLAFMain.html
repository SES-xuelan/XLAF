初始化

  • transfer_func 初始化方法中的回调函数，主要用于旧版本的数据迁移，返回版本号和对象的参数说明，请看例子

Usage:

     --在main.lua中的appyeast.init
appyeast.init(function(last_ver, objs)

  local ver_curr = appyeast.get_sys_value("ver_cur")--当前版本号
  if last_ver == nil then --旧产品升级到appyeast，需要把旧的数据迁移到appyeast中（覆盖安装，保留用户数据）
      if objs["remove"] then appyeast.iap_set_var("remove", objs["remove"]) end
  elseif last_ver~=ver_curr then --版本不同(一般是版本升级，有变量改变则在此处理)

  end

end)

ad_task_show (scene, button_name, ad_tag[, params])
显示 task任务

Parameters:

  • scene the scene obj
  • button_name 按钮名（因为都是点击按钮触发的）
  • ad_tag 广告的位置标志 例如 game levelend 等（注意，buyfree和buyclose有点特殊，具体请看例子）
  • params ad_task_params

Usage:

    appyeast.ad_task_show(scene, "btn_free", "buyfree")
 --绑定到btn_free按钮，并且转到free列表
 --一般是免费获取hint的任务列表，比如：看广告给1个hint、安装某产品给hint等

appyeast.ad_task_show(scene, "btn_close", "buyclose")  (使用此功能需要增加lang，请查看readme)
 --绑定到btn_close按钮，并且显示free列表的第一个任务
 --一般情况下弹窗显示，比如：弹个窗问你是否rate来获得hint

  具体请参考buyhint界面的代码
