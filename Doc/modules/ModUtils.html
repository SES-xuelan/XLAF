Parameters:

  • parent parent group
  • file 图片路径+名字

  • box_height 9宫格的高度

  • insetbottom 下面不被拉伸的大小
  • insetleft 左面不被拉伸的大小
  • opt 可选，其他的参数，包括：imageWidth, imageHeight, return_obj
